Model

Simple enough to argue with.

The dashboard deliberately uses a transparent diffusion model. It is a workbench for questions, not a forecast.

market_share = active / addressable_market
referral_pressure = referrals × market_share × (1 + network_amplification × market_share)
adoption_pressure = (organic_discovery + referral_pressure) × conversion × trust × change_authority
new_agents = (addressable_market - active) × adoption_pressure
weekly_value_per_agent = base_value + min(network_bonus_cap, network_bonus_per_100_agents × active / 100)
weekly_churn = max(min_churn, base_churn - value_churn_relief)
active_next = active + new_agents - active × weekly_churn

Built for playing with Dreamcatcher adoption/value assumptions. No data leaves the browser.
